# Berlin Airport App – UI/UX Case Study

## Overview

This project presents a responsive UI/UX concept for a Berlin Airport mobile application. The goal was to design an intuitive and user-friendly interface that enables passengers to quickly access flight information and airport services on both smartphones and tablets.

## Objectives

* Design a modern and intuitive user interface
* Improve the user experience for travelers
* Create responsive layouts for mobile and tablet devices
* Develop an interactive prototype in Figma

## Features

* Clean and intuitive navigation
* Flight search
* Arrival and departure information
* Flight details
* Responsive layouts for smartphone and tablet
* Interactive prototype with clickable screens

## Design Process

1. User research and analysis of airport applications
2. Information architecture
3. Wireframing
4. High-fidelity UI design
5. Interactive prototyping
6. Usability improvements

## User Flow

* Launch the application
* Search for a flight
* View arrival or departure information
* Open detailed flight information
* Navigate between airport services and flight status

## Color Palette

* Primary: Blue (#0057B8)
* Secondary: White (#FFFFFF)
* Accent: Light Gray (#F5F7FA)
* Status Colors: Green, Orange, Red for flight information

## Typography

* Font Family: Inter
* Clear visual hierarchy
* High readability
* Accessible font sizes for mobile and tablet devices
